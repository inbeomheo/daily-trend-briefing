## 핵심 트렌드 요약
> AI 에이전트 생태계가 "만들기 → 팔기 → 관찰하기"로 가치사슬을 완성하며 SaaS를 빠르게 대체하고 있다. Claude/Anthropic이 범용 AI 1위(Polymarket 100%)를 확정하며 #QuitGPT 운동(250만 명 이탈)과 함께 시장 판도가 재편 중이다. 바이브코딩이 X에서 최대 토픽(140건)을 기록하면서도 "진짜 코딩이 그립다"는 역설적 반발이 동시에 확산되고, Stanford 연구(HN 640pt)는 AI의 아첨 문제를 경고한다. 음성 AI 경쟁이 격화(Cohere STT, Mistral TTS, MS VibeVoice, Gemini Live)되고, 로컬 LLM(Qwen 3.5, Ollama+Open WebUI)은 프라이버시와 비용 절감의 대안으로 부상 중이다. Polymarket은 NVIDIA 시총 1위(100%), Tech 해고 증가(80%), SpaceX IPO(93%)를 예측하며 AI 인프라 투자와 고용 구조 변화의 거시적 방향을 확인시킨다.

### 1. Ollama + Open WebUI로 로컬 LLM 스택 구축 (Reddit 1위 도구)
```bash
# Ollama 설치
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.1:8b

# Open WebUI 실행
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway \
  -v open-webui:/app/backend/data --name open-webui \
  ghcr.io/open-webui/open-webui:main
```
